This is [VideoPong](http://memelab.com.br/projeto/videopong), made by Tiago Pimentel and VJ pixel, from [memeLab](http://memelab.com.br).

VideoPong is a version of [Pong](http://en.wikipedia.org/wiki/Pong) in [augmented reality]().
   
Based on the Pong code by [Alex Mesker](http://www.x37v.com) and on sketches by Tiago Pimentel.
   
This code is made with [Processing](http://processing.org) and is distributed under [GPL](http://www.gnu.org/copyleft/gpl.html).
